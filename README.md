# Syntax
## VARIABLE:
```ps
<==
var string = "value";
==>
```
```ps
<==
var array = ["first", "second", "third"];
==>
```
## DOCTYPE:
```ps
[doctype:type]
```
## ELEMENT, ID, CLASS:
```ps
element {};
```
```ps
element#id {};
```
```ps
element.class {};
```
```ps
element#id.class {};
```
## INPUT:
```ps
tag[input1:"value", input2:"value"]
```
## INTERPOLATION:
```ps
"$/variable/"
```
## LOOPS:
```ps
for (x in y) {};
```
```ps
for (x, i in y) {};
```
## CONDITIONALS:
```ps
if x {};
```
```ps
else {};
```
