# Syntax
## VARIABLE:
```
<==
var string = "value";
==>
```
```
<==
var array = ["first", "second", "third"];
==>
```
## DOCTYPE:
```
[doctype:type]
```
## ELEMENT, ID, CLASS:
```
element {};
```
```
element#id {};
```
```
element.class {};
```
```
element#id.class {};
```
## INPUT:
```
tag[input1:"value", input2:"value"]
```
## INTERPOLATION:
```
"$/variable/"
```
## LOOPS:
```
for (x in y) {};
```
```
for (x, i in y) {};
```
## CONDITIONALS:
```
if x {};
```
```
else {};
```
