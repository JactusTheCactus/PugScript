# Syntax
## VARIABLE:
```
<==
var string = "value";
==>
```
```
<==
var array = ["first", "second", "third"];
==>
```
```
<==
var object = /<num1: "first", num2: "second", num3: "third">/;
==>
```
## EXTERNAL DATA:
```
<==
var data from "data.json";
==>
```
## DOCTYPE:
```
[doctype:type]
```
## ELEMENT, ID, CLASS:
```
element {};
```
```
element#id {};
```
```
element.class {};
```
```
element#id.class {};
```
## INPUT:
```
element[input1:"value", input2:"value"] {};
```
## INTERPOLATION:
```
"$/variable/"
```
## LOOPS:
```
for (x in y) {};
```
```
for (x, i in y) {};
```
## CONDITIONALS:
```
if x {};
```
```
else {};
```
